# Rutina JiuJitsu - Tracker App 🥋

Una aplicación web diseñada específicamente para atletas de JiuJitsu Brasileño (BJJ). Esta herramienta te permite organizar, registrar y medir tus entrenamientos de fuerza, movilidad y acondicionamiento físico de forma paralela a tus clases de tatami.

## 🌟 Características Principales
* **Planificación Semanal:** Organización visual de tus días de Fuerza (Full Body), JiuJitsu y Recuperación.
* **Biblioteca de Ejercicios:** Rutinas enfocadas en movimientos clave para el grappling (empuje, tracción, potencia, core y agarre).
* **Registro de Progreso:** Gráficas automáticas que calculan tu volumen levantado por sesión y tu mejora en el agarre (esencial para el control de solapas).
* **Temporizador Inteligente:** Temporizadores integrados para descansos y circuitos de cardio (como tolerancia al lactato).
* **Diseño Oscuro (Dark Mode):** Interfaz elegante y amigable para la vista, diseñada simulando una aplicación nativa de iOS.

## 📱 Cómo usarla en tu iPhone como una App
Para disfrutar de la mejor experiencia (pantalla completa sin barra de direcciones):
1. Abre el enlace de la aplicación desde **Safari** en tu iPhone.
2. Toca el botón de **Compartir** en la barra inferior (el cuadrado con la flecha hacia arriba).
3. Desliza hacia abajo en el menú y selecciona **"Agregar a Inicio"** (Add to Home Screen).
4. Confirma tocando **"Agregar"**. ¡Listo! Ahora tendrás el ícono en tu pantalla principal.
